## Deskripsi Tugas
Aplikasi CRUD (Create, Read, Update, Delete) sederhana menggunakan Jetpack Compose yang mengimplementasikan:
- Navigation Component untuk navigasi antar layar.
- ViewModel untuk manajemen state dan logika bisnis.
- Berbagai komponen input Jetpack Compose:
  - `OutlinedTextField` (Teks)
  - `RadioButton` (Pilihan Tunggal - Prioritas)
  - `ExposedDropdownMenu` (Pilihan Dropdown - Kategori)
  - `Switch` (Boolean - Urgent)
  - `Slider` (Progress - Persentase)
  - `Checkbox` (Pilihan Banyak - Label)
